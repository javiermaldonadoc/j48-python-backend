# j48-python-backend

Private staging repository for the J48-targeting Python backend extracted from the IDEA workspace.

Current scope:
- strict J48-oriented core
- scikit-learn-compatible wrapper
- validation-oriented engineering baseline

Data policy:
- IDS datasets are not redistributed in this repository
- public datasets are referenced from their original sources
- reproduction scripts should assume user-provided datasets when redistribution is restricted

Release policy:
- repository starts private
- repository will become public only after packaging, documentation, and legal review are complete
