# Data Availability

## Repository policy

This repository does not redistribute the datasets used in the broader research workflow.

## Public datasets

Public non-IDS datasets used in validation should be obtained from their original public sources.

The repository may reference those sources and expected input formats, but it does not need to mirror the datasets locally.

## IDS datasets

IDS datasets used in the paper are not redistributed in this repository.

They originate from prior sources for which the authors do not currently claim redistribution rights in this derived software package.

## Reproduction expectation

When a validation or reproduction step depends on external datasets, users are expected to:

1. obtain the datasets from the original sources,
2. place them locally,
3. adapt paths or manifests as needed for the released workflow.

## Purpose of this statement

This statement is included to make the repository's legal and practical scope explicit for users, reviewers, and editors.
